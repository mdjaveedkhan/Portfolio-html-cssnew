YouTube Video Downloader


This project leverages the yt-dlp library to fetch and download videos in specific resolutions with audio, providing users with an easy way to download their preferred video quality.

Key Features:

  • User Interface: Designed a simple and functional GUI using Tkinter.
  • Video Quality Options: Supports video downloads in 1080p, 720p, and 480p with audio.
  • Customizable Download Path: Allows users to select their preferred download location.
  • Integrated Notifications: Provides user-friendly notifications for successful downloads and error handling.
    • This project enhanced my understanding of Python's GUI libraries, file handling, and working with external APIs for video processing. It also involved handling video formats and streamlining the user experience for downloading media from YouTube.
